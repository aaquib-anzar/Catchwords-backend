# 🎯 Catchwords.AI — Backend

AI-powered caption generator for social media. Generate optimized captions for Instagram, Twitter, LinkedIn and more — with full control over tone, language, and length.

🌐 **Live Demo:** [catchwords-frontend-ecru.vercel.app](https://catchwords-frontend-ecru.vercel.app/)
💻 **Frontend Repo:** [Catchwords-frontend](https://github.com/aaquib-anzar/Catchwords-frontend)

---

## 🚀 Features

- 🤖 AI-powered caption generation using Gemini API
- 📱 Supports multiple platforms — Instagram, Twitter, LinkedIn and more
- 🎨 Customize tone (casual, professional, funny, inspirational)
- 🌍 Multi-language support
- 📏 Control caption length
- 🔐 JWT based authentication
- 📝 Caption history saved per user

---

## 🛠️ Tech Stack

- **Runtime:** Node.js
- **Framework:** Express
- **Database:** MongoDB + Mongoose
- **AI:** Google Gemini API
- **Auth:** JWT
- **Deployment:** Render
